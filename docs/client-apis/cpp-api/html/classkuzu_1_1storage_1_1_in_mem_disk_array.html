Detailed Description

template<typename T>
class kuzu::storage::InMemDiskArray< T >

Stores an array of type U's page by page in memory, using OS memory and not the buffer manager. Designed currently to be used by lists headers and metadata, where we want to avoid using pins/unpins when accessing data through the buffer manager.
