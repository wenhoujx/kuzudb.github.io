12// A DataChunk represents tuples as a set of value vectors and a selector array.
13// The data chunk represents a subset of a relation i.e., a set of tuples as
14// lists of the same length. It is appended into DataChunks and passed as intermediate
15// representations between operators.
16// A data chunk further contains a DataChunkState, which keeps the data chunk's size, selector, and
17// currIdx (used when flattening and implies the value vector only contains the elements at currIdx
18// of each value vector).
