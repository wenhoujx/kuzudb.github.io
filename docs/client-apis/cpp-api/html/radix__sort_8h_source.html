23// RadixSort sorts a block of binary strings using the radixSort and quickSort (only for comparing
24// string overflow pointers). The algorithm loops through each column of the orderByVectors. If it
25// sees a column with string, which is variable length, it will call radixSort to sort the columns
26// seen so far. If there are tie tuples, it will compare the overflow ptr of strings. For subsequent
27// columns, the algorithm only calls radixSort on tie tuples.
